## 🧠 Core Idea

General Relativity and Quantum Field Theory possess a **structural asymmetry**:

- QFT requires a fixed background to define locality.  
- GR denies any fixed background and makes the metric dynamical.

The **Gravitational Baseline Principle** resolves this mismatch by treating gravity not as a peer interaction, but as the **universal geometric baseline** emerging from a quantum information substrate.

The numerical results in this repository show that geometry, coordinates, and a massless graviton arise naturally from entanglement structure alone.
